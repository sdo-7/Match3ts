```
git clone git@github.com:sdo-7/Match3ts.git
cd Match3ts
```

To launch with Docker
```
docker compose up --build
```
Open via web browser localhost:3000

To launch with npm
```
npm install
npm run compile
npm exec http-server build
```
Open via web browser localhost:8080
